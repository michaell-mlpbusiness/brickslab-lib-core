# BricksLab — Documentation

Cette documentation sert à :
- démarrer rapidement après clone,
- contribuer à la librairie (web/mobile),
- comprendre le contrat de tokens + theming,
- ajouter un composant et l’exposer dans le catalogue.

## Entrées rapides
- Démarrage : `getting-started.md`
- Workflow contribution : `workflow.md`
- Tokens & thèmes : `tokens/token-contract.md` + `tokens/theming.md`
- Ajouter un composant : `components/add-component.md`
- L’ajouter au catalogue : `catalog/add-to-catalog.md`
- Checklist qualité : `quality/checklist.md`
- Dépannage : `quality/troubleshooting.md`
